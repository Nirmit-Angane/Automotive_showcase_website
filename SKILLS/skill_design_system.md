# SKILL: Design System (Brutalist Motorsport)

## Color Palette
- **Primary Black**: `#0A0A0A` (Backgrounds, heavy elements)
- **Racing Red**: `#DC0000` (Accents, CTAs, aggressive highlights)
- **Concrete Gray**: `#3D3D3D` (Borders, secondary text, structural lines)
- **Warning Yellow**: `#FFD600` (Alerts, badges, technical data)
- **Pure White**: `#FFFFFF` (Primary text, high contrast elements)

## Typography
- **Headings**: `Bebas Neue` (Ultra-condensed, uppercase, massive scale)
- **Body**: `Space Grotesk` (Geometric, technical, legible)
- **Micro/Data**: `Chakra Petch` (Tech-inspired, angular)

## Spacing & Grid
- **Base Unit**: 4px
- **Container**: Fluid with max-width 1920px (Cinematic)
- **Margins**: Aggressive negative space (e.g., `ml-[-20vw]`) to break the grid.

## Visual Rules
- **No Rounded Corners**: `rounded-none` everywhere.
- **Hard Shadows**: `shadow-[4px_4px_0px_#DC0000]`.
- **Borders**: Thicc `border-2`.
- **Overlays**: Multiply/Hard-light blend modes for images.
