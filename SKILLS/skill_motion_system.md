# SKILL: Motion System

## Framer Motion Defaults
- **Transition**: `type: "spring", stiffness: 300, damping: 30` (Snappy, mechanical)
- **Hover**: Startlingly fast color invert or displacement.
- **Scroll**: Parallax on distinct layers (text moves faster than image).

## Interactions
- **Magnetic Buttons**: Elements that snap to cursor.
- **Text Reveal**: Staggered character reveal for headers.
- **Image Reveal**: Clip-path animations (diagonal wipe).
