# Premium Automotive Showcase - Brutalist Motorsport Edition

## Project Overview
A premium multi-brand automotive showcase website featuring Ferrari, Lamborghini, McLaren, and Porsche. This project implements a **Brutalist Motorsport** design direction, prioritizing raw industrial aesthetics, aggressive typography, and cinematic visuals.

## Design Philosophy: Brutalist Motorsport
- **Aesthetic**: Raw power, construction-site industrialism, technical precision.
- **Palette**: Primary Black (`#0A0A0A`) & Racing Red (`#DC0000`).
- **Typography**: `Bebas Neue` (Headers) & `Space Grotesk` (Body).
- **Core Rules**: No rounded corners, no soft gradients, aggressive asymmetry.

## Tech Stack (Mandatory)
- **Frontend**: Next.js 14+ (App Router), TypeScript
- **Styling**: Tailwind CSS, Shadcn UI
- **Animation**: Framer Motion (60fps mechanical transitions)
- **Deployment**: Vercel
